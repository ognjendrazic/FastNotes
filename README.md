FastNotes is a simple personal note-taking app built with React Native and Expo.
The app allows users to create, view, and read notes locally (in-memory).

## Dependencies
- Node.js (v18+ recommended)
- React Native
- Expo

## Install
npm install

## Run (development)
npx expo start

Scan the QR code with Expo Go on a physical device, or run using an emulator.
